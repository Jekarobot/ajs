# 🚀 AJS

**AJS** — это легковесный JavaScript-проект, предназначенный для демонстрации базовой настройки сборки с использованием Webpack. Проект включает минимальную структуру для разработки и сборки модульного кода.

## 🛠️ Технологический стек

- **JavaScript (ES6+)** — основной язык разработки
- **Webpack** — сборщик модулей
- **Node.js** — среда выполнения

## 📦 Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/your-username/ajs.git
   cd ajs
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Запустите сборку:
   ```bash
   npm run build
   ```

4. Для разработки с горячей перезагрузкой:
   ```bash
   npm run dev
   ```

## 💻 Пример использования

После сборки проекта, подключите скомпилированный файл в HTML:

```html
<script src="dist/index.js"></script>
```

Или используйте в Node.js:

```javascript
const ajs = require('./dist/index.js');
console.log(ajs); // Вывод результата работы модуля
```

## 📁 Структура проекта

```
ajs/
├── dist/               # Скомпилированные файлы
│   └── index.js        # Выходной файл сборки
├── src/                # Исходный код
│   └── index.js        # Главный модуль
├── package.json        # Конфигурация npm
├── package-lock.json   # Фиксация зависимостей
└── webpack.config.js   # Конфигурация Webpack
```
